# <a name="power-bi-destination"></a>Power BI hedefi

[!include [banner](../includes/banner.md)]

Giden belgeler oluşturmak üzere yapılandırılan bir Elektronik raporlama (ER) biçiminin her klasör veya dosya bileşeni için bir Microsoft Power BI hedefi yapılandırabilirsiniz. Hedefin ayarın bağlı olarak, oluşturulan bir belge daha önce yapılandırılmış olan bir SharePoint klasöründe depolanır.

Verilerinizi Dynamics 365 Finance örneğinizden Microsoft Power BI hizmetlerine aktarmak amacıyla düzenlemek için Elektronik raporlama (ER) yapılandırmanızı kullanmak üzere **Etkin**'i **Evet** olarak ayarlayın. Aktarılan dosyalar bu amaçla yapılandırılmış bir Microsoft SharePoint Server örneği üzerinde depolanmalıdır. Daha fazla bilgi için bkz. [Power BI'a veri çekmek için Elektronik raporlamayı (ER) yapılandırma](general-electronic-reporting-report-configuration-get-data-powerbi.md).
