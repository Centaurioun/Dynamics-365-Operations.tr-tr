# <a name="LEN">LEN ER işlevi</a>

[!include [banner](../includes/banner.md)]

Bu `LEN` işlev, belirtilen dizedeki karakter sayısını bir *tamsayı* değeri olarak döndürür.

## <a name="syntax"></a>Sözdizimi

```vb
LEN (text)
```

## <a name="arguments"></a>Bağımsız değişkenler

`text`: *Dize*

Metni temsil eden bir *dize* değeri.

## <a name="return-values"></a>Dönüş değerleri

*Tamsayı*

Sonuç sayısal değeri.

## <a name="example"></a>Örnek

`LEN ("Sample")`, **6** döndürür.
