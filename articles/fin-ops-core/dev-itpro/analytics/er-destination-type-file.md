# <a name="file-destination"></a>Dosya hedefi

[!include [banner](../includes/banner.md)]

Giden belgeler oluşturmak üzere yapılandırılan bir Elektronik raporlama (ER) biçiminin her KLASÖR veya DOSYA bileşeni için bir dosya hedefi yapılandırabilirsiniz. Hedefin ayarına bağlı olarak, oluşturulan belge web tarayıcısından indirilebilir.

**Hedef ayarları** sayfasında **Etkin**'in ayarını **Evet** yaparsanız, yapılandırmanın çalıştırılması bittiğinde bir açma ya da kaydetme iletişim kutusu açılır.
