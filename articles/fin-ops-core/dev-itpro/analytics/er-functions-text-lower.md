# <a name="lower-er-function"></a>LOWER ER işlevi

[!include [banner](../includes/banner.md)]

Bu `LOWER` işlev, küçük harflere dönüştürüldükten sonra bir *dize* değeri olarak belirtilen metin dizesini döndürür.

## <a name="syntax"></a>Sözdizimi

```vb
LOWER (text)
```

## <a name="arguments"></a>Bağımsız değişkenler

`text`: *Dize*

Metni temsil eden bir *dize* değeri.

## <a name="return-values"></a>Dönüş değerleri

*Dize*

Sonuç metin değeri.

## <a name="example"></a>Örnek

`LOWER ("Sample")`, **"Örnek"** döndürür.
