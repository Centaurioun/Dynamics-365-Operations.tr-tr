# <a name="supported-web-browsers-for-lifecycle-services"></a>Lifecycle Services için desteklenen web tarayıcıları 

[!include[banner](../includes/banner.md)]

Kullanıcılar, aşağıdaki popüler tarayıcıların en son sürümlerini kullanarak Microsoft Dynamics Lifecycle Services'e (LCS) erişebilir: 

- Microsoft Edge (önerilen: [Chromium-based Edge](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

Her web tarayıcısı için en son sürümü bulmak için, yazılım üreticisinin web sitesine gidin. En iyi performans ve deneyim için modern bir tarayıcının en son sürümünü, özellikle de Microsoft Edge'i kullanmanızı öneririz.
