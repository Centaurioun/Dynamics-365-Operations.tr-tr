# <a name="microsoft-partner-sales-and-marketing-learning-catalog"></a>Microsoft İş Ortağı Satış ve Pazarlama Öğrenme Kataloğu

Müşterilerinizin doğru çözümü satın almasından yardımcı olacak mı?
