# <a name="skill-level-type"></a>Yetenek düzeyi türü


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Bu makalede, Dynamics 365 Human Resources için Yetenek düzeyi türü seçenek kümesi açıklanmaktadır.

Fiziksel ad: mshr_hrmskillleveltype

Bu numaralandırma, bir adayın yeteneklerine atanan düzeyi kategorize eden bir seçenek kümesi sağlar.

| Değer | Etiket | Tanım |
| --- | --- | --- |
| 200000000 | Fiili değer | Gerçek yetenek düzeyi. |
| 200000001 | Hedef | Yetenek düzeyi için hedeflenen değer. |
