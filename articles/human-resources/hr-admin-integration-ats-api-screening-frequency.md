# <a name="screening-frequency"></a>Filtreleme sıklığı

Bu konu, Dynamics 365 Human Resources için filtreleme sıklığı seçenek kümesini açıklamaktadır.

Fiziksel ad: mshr_hcmfrequencyunit

Bu sabit listesi, filtreleme sıklığı için değer seçenek kümesi sağlar. 

| Değer | Etiket | Tanım |
| --- | --- | --- |
| 200000000 Tek seferlik | Tarama sadece bir kez gereklidir. |
| 200000001 Günlük | Filtreleme sıklığı gün cinsinden hesaplanır. |
| 200000002 Hafta | Filtreleme sıklığı hafta cinsinden hesaplanır. |
| 200000003 Ay | Filtreleme sıklığı ay cinsinden hesaplanır. |
| 200000004 Yıl | Filtreleme sıklığı yıl cinsinden hesaplanır. |
