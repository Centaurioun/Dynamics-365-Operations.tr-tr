Bu numaralandırma, Human Resources sanal varlıklarında işlem türü için seçenek kümesi sağlar.

| Değer | Etiket | Tanım |
| --- | --- | --- |
| 200000000 | Hiçbiri | Hiçbiri. |
| 200000001 | Yıllık | Yıllık. |
| 200000002 | Yıldönümü | Yıldönümü. |
| 200000003 | Döngüsel değil | Döngüsel değil. |
| 200000004 | ½ yıl | Yarım yıllık. |
| 200000005 | Üç Aylık | Üç Aylık. |
| 200000006 | Monthly | Aylık. |
