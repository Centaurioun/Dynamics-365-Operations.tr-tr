## <a name="option-set-process-type"></a>Seçenek kümesi: İşlem türü

Fiziksel ad: mshr_hrmCompProcessType

Bu numaralandırma, Human Resources sanal varlıklarında işlem türü için seçenek kümesi sağlar.

| Değer | Etiket | Tanım |
| --- | --- | --- |
| 200000000 | Hiçbiri | Hiçbiri. |
| 200000001 | Seçili | Seçili. |
| 200000002 | Silindi | Silindi. |
