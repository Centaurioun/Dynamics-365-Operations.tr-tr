# <a name="marital-status"></a>Medeni durum

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Bu konu, Dynamics 365 Human Resources için Medeni durum seçenek kümesini açıklar.

Fiziksel ad: mshr_dirpersonmaritalstatus

Bu numaralandırma, kişi kayıtlarında medeni durum için seçenek kümesi sağlar.

| Değer | Etiket | Tanım |
| --- | --- | --- |
| 200000000 | Hiçbiri | Hiçbir değer seçilmedi.
| 200000001 | Bekar | Bekar. |
| 200000002 | Evli | Evli. |
| 200000003 | Boşanmış | Boşanmış. |
| 200000004 | Dul | Dul. |
