# <a name="applicant-integration-result"></a>Başvuran tümleştirmesi sonucu

Bu konu, Dynamics 365 Human Resources için Başvuran tümleştirme sonucu seçenek kümesini açıklar.

Fiziksel ad: mshr_hcmapplicantintegrationresult

Bu numaralandırma aday kaydı durumunu sağlar.

| Değer | Etiket | Tanım |
| --- | --- | --- |
| 200000000 | İşlenmedi | Aday hala değerlendirilmektedir. |
| 200000001 | İşe alındı | Aday işe alındı. |
| 200000002 | İşe alınmadı | Adayı işe almama kararı verildi. |
| 200000003 | Bırakıldı | Aday değerlendirmeden çıkarıldı. |
