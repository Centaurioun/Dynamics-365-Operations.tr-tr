# <a name="create-employment-categories"></a>İstihdam kategorileri oluşturma

Dynamics 365 Human Resources'ta iş kategorilerini, çalışanlarınıza uyacak şekilde ayarlayın.

1. **Sosyal haklar** yönetimi çalışma alanında, **Kur** altında, **İstihdam kategorileri**'nı seçin.

2. **Yeni**'yi seçin.

3. **Çalışan türü**, **istihdam kategorisi** ve **açıklaması** değerlerini belirtin.

4. **Kaydet**'i seçin.
