# <a name="gender"></a>Cinsiyet


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Bu makalede, Dynamics 365 Human Resources için Cinsiyet seçenek kümesi açıklanmaktadır.

Fiziksel ad: mshr_hcmpersongender

Bu numaralandırma, aday için cinsiyet seçenek kümesi sağlar. Bu, mshr_hcmpersongender seçenek kümesinde bulunur.

| Değer | Etiket | Tanım |
| --- | --- | --- |
| 200000000 | Hiçbiri | Cinsiyet belirtilmedi. |
| 200000001 | Erkek | Erkek. |
| 200000002 | Kadın | Kadın. |
| 200000003 | NonSpecific | Belirli bir cinsiyet seçimi olmaması. |
