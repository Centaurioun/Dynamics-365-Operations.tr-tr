Bu makalede, Dynamics 365 Human Resources için İstihdam türü seçenek kümesi açıklanmaktadır.

## <a name="option-set-employment-type"></a>Seçenek kümesi: İstihdam türü

Bu numaralandırma, çalışan için istihdam türü seçenek kümesi sağlar. Bu, mshr_hcmemploymenttype seçenek kümesinde bulunur.

| Değer | Etiket | Tanım |
| --- | --- | --- |
| 200000000 | Personel | Personel |
| 200000001 | Yüklenici | Yüklenici |
