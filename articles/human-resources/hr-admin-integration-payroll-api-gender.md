# <a name="payroll---gender-option-set"></a>Bordro - Cinsiyet seçenek kümesi


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Bu konu, Dynamics 365 Human Resources için Cinsiyet seçenek kümesini açıklar.

## <a name="option-set-gender-option"></a>Seçenek kümesi: Cinsiyet seçeneği

Bu numaralandırma, çalışan için cinsiyet seçenek kümesi sağlar. Bu, mshr_hcmpersongender seçenek kümesinde bulunur.

| Değer | Etiket | Tanım |
| --- | --- | --- |
| 200000000 | Hiçbiri | Cinsiyet belirtilmedi. |
| 200000001 | Erkek | Erkek. |
| 200000002 | Kadın | Kadın. |
| 200000003 | NonSpecific | Belirli bir cinsiyet seçimi olmaması. |
