# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Adres defterinde bir çalışan adını değiştirdikten sonra kaynak adı güncelleştirilmiyor

## <a name="symptoms"></a>Belirtiler

Genel adres defterinde bir çalışan adını değiştirirseniz eşleşen kaynak adı, ana kaynak grubunda güncelleştirilmez.

## <a name="resolution"></a>Çözüm

Eşleşen kaynak adı şu anda otomatik olarak güncelleştirilmez. Sorunu gidermek için kaynak adını el ile güncelleştirmeniz gerekir.
