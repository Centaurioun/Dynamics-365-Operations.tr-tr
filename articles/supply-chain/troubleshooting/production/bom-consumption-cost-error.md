# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>Ürün reçetesinin consumptionCost değeri hatası nedeniyle üretim siparişi sonlandırılamıyor

## <a name="symptoms"></a>Belirtiler

Üretim siparişini sonlandırmayı denediğinizde aşağıdaki hata iletisini alırsınız:

> Ürün reçetesinin consumptionCost değerinin hesaplanması, stoktan çıkıştan sonra negatif olmalıdır.

## <a name="resolution"></a>Çözüm

Bu sorun 10.0.15 sürümünde giderilmiştir.
