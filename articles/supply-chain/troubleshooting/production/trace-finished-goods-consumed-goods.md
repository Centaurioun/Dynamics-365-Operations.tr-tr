# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>Mamul malların seri numaraları tüketilen malların seri numaralarıyla karşılaştırılıp izlenemiyor

## <a name="symptoms"></a>Belirtiler

Mamul malların seri numaralarını, üretim emrinin bu mamul malları üretmek için tükettiği malzemelerin seri numaralarıyla karşılaştırıp izleyemezsiniz.

## <a name="workaround"></a>Geçici çözüm

Geçici çözüm olarak miktar 1 olacak şekilde üretim emirleri oluşturun.
