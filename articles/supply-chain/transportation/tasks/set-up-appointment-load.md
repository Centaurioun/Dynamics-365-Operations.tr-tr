# <a name="set-up-an-appointment-for-a-load"></a>Yük için randevu ayarlama

[!include [task guide banner](../../includes/task-guide-banner.md)]

Bu yordam bir yük için giriş/çıkış noktası randevusunun nasıl ayarlanacağını ve planlanacağını gösterir. Bu genellikle taşımacılık düzenleyicisi tarafından yapılır. Bu yordamı USMF demo veri şirketinde kullanabilirsiniz.


## <a name="select-the-load"></a>Yükü seçme
1. Taşıma yönetimi > Planlama > Yük planlama çalışma ekranına gidin.
2. Sevk edileni ve teslim alınanı gizle onay kutusunu temizleyin.
3. Listede durumu Sevk Edildi olan yükü seçin.
4. Taşıma'ya tıklayın.
5. Randevu zamanı planlama'ya tıklayın.

## <a name="create-an-appointment"></a>Randevu oluşturma
1. Yeni'ye tıklayın.
2. Randevu kuralı alanına bir değer girin veya seçin.
3. Kaydet'e tıklayın.
4. Durumu güncelleştir'e tıklayın.
5. Kesinleştir öğesine tıklayın.
6. Kaydet'e tıklayın.
7. Sayfayı kapatın.
