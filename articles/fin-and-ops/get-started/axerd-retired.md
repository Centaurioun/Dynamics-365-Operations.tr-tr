# <a name="axerd-retired"></a>AxErd kaldırıldı

Ne yazık ki bir çok Microsoft Dynamics AX 2012 tablosu için Varlık ilişkisi şemaları (ERD'ler) sağlayan AxErd web sayfasını kaldırmak zorunda kaldık. Projenin kaynak kodunu ve web sayfasının HTML dosyalarını GitHub havuzuna nakledebildik: https://github.com/Microsoft/ax-2012-doc-tools

HTML sayfaları havuzda, [https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Default.htm](https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Default.htm) konumundadır

Kendi ERD'lerinizi oluşturma yönteminin açıklaması şu konumdadır: [https://github.com/Microsoft/ax-2012-doc-tools/blob/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Help-HowToUse-AxErd.htm](https://github.com/Microsoft/ax-2012-doc-tools/blob/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Help-HowToUse-AxErd.htm)

Çözüm AxErd klasörünün kökündedir [https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd](https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd).
